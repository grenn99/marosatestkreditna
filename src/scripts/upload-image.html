Instructions

This tool will help you prepare the image for the "Darilni paket" product:

  1. Select an image file (preferably a gift package or gift box image)
  2. The image will be renamed to gift_package.jpg
  3. Download the image and place it in the /public/images/darilni_paket/ folder on your server
  4. Then run the fix-image.html script to update the database
